## 下载（三步）

1. 点击本页右上方绿色的 **Code** 按钮
2. 选择 **Download ZIP**
3. 解压后，先打开 **《零基础上手教程.txt》**——从装软件到出第一张图，一步一图，完全没碰过命令行也能跟着走完
